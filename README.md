# book-tracker-app
Project Title: Book Tracker

Description: A personal reading log application designed for students and book lovers to organize their libraries. The app allows users to catalog books they have read or plan to read, maintain personal notes on each title, and track their reading status (e.g., "Want to Read," "Reading," "Finished").
Target Users: Individual readers looking for a minimalist tool to manage their reading lists without complex social media features.

Technical Stack
Stack 1 (MERN)
- Frontend (FE): React + Vite
- Backend (BE): Node.js + Express.js
- Database (DB): MongoDB Atlas 
- Tools: VS Code, Git/GitHub, Postman (for API testing).

Functional Requirements
 - [ ] Register and log in.
 - [ ] Add a new book.
 - [ ] Edit book details.
 - [ ] Delete a book.
 - [ ] Change reading status.
 - [ ] Add notes to a book.
 - [ ] Search and filter books.
 - [ ] Track reading progress.
