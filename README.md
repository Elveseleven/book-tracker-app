# book-tracker-app
A reading log where users can add books they’ve read or plan to read. Includes personal notes, reading status, and progress visualization.

Technology Stack (MERN)
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Environment: Docker (planned for future milestone)

Functional Requirements
- [ ]Add a Book: User can input a title and author to save a new book to the database.
- [ ]Display Books: The app retrieves and shows a list of all saved books from MongoDB.
- [ ]Delete a Book: User can remove a book from their list
- [ ]Update Status: User can change a book's status (e.g., from "Reading" to "Finished").
